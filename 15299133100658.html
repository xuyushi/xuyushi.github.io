使用命令行查看android sqlite数据库

2015/6/5 posted in  Android

首先adb shell连接虚拟机或者设备

$ adb shell

进入数据库目录

通常在/data/data/包名/databases 下

$ cd /data/data/com.example.xuyushi.loadertest/databases

进入数据库

使用sqlite3 进入，需要安装工具

$ sqlite3 xxxx

提示

SQLite version 3.7.11 2012-03-20 11:35:50
Enter ".help" for instructions
Enter SQL statements terminated with a ";"
sqlite>

使用.help可以查看帮助

使用.table查看已建的表格

$ .table

提示

sqlite> .table
STUDENT           android_metadata

使用SELECT查看表内容

$ SELECT * FROM STUDENT;

注意有分号

$ sqlite> SELECT * FROM STUDENT;
1|zhangsan|22
