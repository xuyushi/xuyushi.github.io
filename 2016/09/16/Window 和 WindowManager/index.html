Window 和 WindowManager

发表于 2016-09-16 | 分类于 android

概述

window

  • window 是一个抽象类，具体实现是PhoneWindow
  • window 也是一个抽象的概念，每个 window 内对应这一个 DecorView 和一个 ViewRootImpl，window 和DecorView 通过ViewRootImpl联系。

    WindowManager

    WindowManager 是 外界访问 Window 的入口

    WindowManagerService

    window 的具体实现位于WindowManagerService中，WindowManager和WindowManagerService的交互是一个 IPC 的过程，
