配置单元测试

首先，确认在Build Variants窗口内的Test Artifact中选择了”Unit Tests”。

然后，在工程的src文件夹内创建test和test/java文件夹。需要注意的是，你不能在Android视图下进行这些操作，要么在系统的文件管理器内创建，要么在工程窗口左上方点击下拉菜单选择Project视图。最终的工程结构应该是这样的：
